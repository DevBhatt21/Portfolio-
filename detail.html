My Projects

Project One - Angry Bird Project Using Java

Summary: I developed an Angry Birds-inspired game in Java featuring physics-based projectile motion, collision detection, and interactive gameplay. The game includes a slingshot mechanism, destructible structures, a scoring system, and immersive sound effects. Key technologies used include Java Swing/JavaFX, a custom physics engine, event handling, and animation. Challenges such as projectile physics, collision handling, and game loop optimization were addressed efficiently. Future improvements include level progression, enhanced graphics, and additional interactive elements. This project highlights my problem-solving skills, game development expertise, and Java programming proficiency.

Project Two - UK Election Analysis Using Python

I developed a Python-based election analysis tool for the recent UK elections, providing 13 user-selectable options to explore results for candidates, constituencies, and specific queries. The project integrates data processing, visualization, and interactive user input to present election insights effectively. Key features include filtering by candidate or constituency, statistical summaries, and graphical representations of results. The program demonstrates my skills in data analysis, Python scripting, and user-focused application design.

This project highlights my ability to create professional, user-friendly web solutions while aligning with business needs.
